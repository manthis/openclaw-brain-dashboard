## How to Contribute Screenshots

If you're running the dashboard and want to contribute screenshots:

1. Take high-quality screenshots (1920x1080 or higher)
2. Ensure no sensitive data is visible (tokens, passwords, personal info)
3. Save as PNG with descriptive filename
4. Submit via PR to this repository

## Guidelines

- **Resolution:** Minimum 1920x1080, PNG format
- **Theme:** Show both light and dark mode when possible
- **Annotations:** Add arrows/highlights for key features (optional)
- **Privacy:** Blur or redact any sensitive information
